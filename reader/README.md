# Wenyi 局域网阅读器

这是一个只读 Wenyi `state/<书名>` 状态目录的移动端网页阅读器。它不会修改、锁定或删除翻译状态，也不会调用模型。

## 启动

在 Wenyi 项目根目录运行：

```powershell
uv run python reader\app.py "state\屈曲ラヴァー_身を滅ぼしてしまいそうな初恋"
```

默认监听 `0.0.0.0:8765`。终端会打印本机和局域网访问地址；手机与电脑连接同一局域网后，在手机浏览器中打开局域网地址即可。

如果 Windows 防火墙弹出网络访问询问，只允许可信的“专用网络”即可。不要在不可信公共网络上开放此服务。

自定义监听地址或端口：

```powershell
uv run python reader\app.py "state\书名" --host 127.0.0.1 --port 9000
```

## 页面行为

- 默认暗色底、柔和白字，适合手机阅读。
- 打开或切换章节时读取该章当前 JSON 译文。
- 每 60 秒查询一次全书翻译与审校状态。
- “立即查询”按钮可随时手动查询。
- 只有选中章节的 JSON 发生变化时才重新载入正文，并尽量保持滚动位置。
- `target` 为空的未翻译段落不会显示。
- “显示原文”可以在当前译文下展开对应日文，方便随时对照；默认关闭。
- 章节位置和正文字号保存在手机浏览器本地。

`review_status: pending` 在未启用最终审校时显示为“未审校”，不影响阅读或 EPUB 输出。
